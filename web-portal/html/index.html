yepui

基于 Vue.js 的移动端组件库

开始使用 查看示例

用法

# 安装nrm npm install -g nrm # 添加jdnpm源 nrm add jdnpm2 http://registry.m.jd.com/ nrm use jdnpm2 # 安装组件 npm install --save @jdcfe/yepui
// 引入全部组件 import yepui from '@jdcfe/yepui'; Vue.use(yepui); // 按需引入部分组件 import {toast} from '@jdcfe/yepui';

特性介绍

  • yepui 包含丰富的 CSS 和 JS 组件，能够满足日常的移动端开发需要。通过它，可以快速构建出风格统一的页面，提升开发效率。
  • 真正意义上的按需加载组件。可以只加载声明过的组件及其样式文件，无需再纠结文件体积过大。
  • 考虑到移动端的性能门槛，yepui 采用 CSS3 处理各种动效，避免浏览器进行不必要的重绘和重排，从而使用户获得流畅顺滑的体验。
  • 依托 Vue.js 高效的组件化方案，yepui 做到了轻量化。即使全部引入，压缩后的文件体积也仅有 ~30kb (JS + CSS) gzip。
