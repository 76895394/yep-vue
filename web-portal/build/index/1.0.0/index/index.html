YepUI

YepUI 是一款基于 Vue.js 2.0 的前端 UI 组件库，主要用于快速搭建移动端网站

开始使用 查看示例

特性

  • 1. 基于 Vue 开发的 UI 组件
  • 2. 使用 npm + webpack + babel 的工作流，支持 ES2015
  • 3. CSS 样式独立，即使使用不同的框架实现，也都能保持统一的 UI 风格提供友好的 API，可灵活的使用组件

用法

# 安装nrm npm install -g nrm # 添加jdnpm源 nrm add jdnpm2 http://registry.m.jd.com/ nrm use jdnpm2 # 安装组件 npm install --save @jdcfe/YepUI
// 引入全部组件 import YepUI from '@jdcfe/YepUI'; Vue.use(YepUI); // 按需引入部分组件 import {yepToast} from '@jdcfe/YepUI';

功能介绍

  • 流畅的体验

    考虑到移动端的性能门槛， 于是采用 CSS3处理各种动效，避免浏览器进行不必要的重绘和重排，从而使用户获得流畅顺滑的体验。

  • 丰富的组件

    包含丰富的 CSS 和 JS 组件，能够满足日常的移动端开发需要。通过它，可以快速构建出风格统一的页面，提升开发效率。

  • 轻量化

    依托 Vue.js 高效的组件化方案， 做到了轻量化。即使全部引入，压缩后的文件体积也仅有 ~30kb(JS + CSS) gzip。

  • 按需加载

    真正意义上的按需加载组件。可以只加载声明过的组件及其样式文件，无需再纠结文件体积过大。
